SnipType is a Firefox extension for fast text snippet expansion using typed shortcuts.

Type `/hi` ‚Üí get predefined text instantly.

Why not Text Blaze? Because it works in Chrome, not Firefox, and i did not find anything like it

---

## Features

- [x] Shortcut-based text expansion
- [x] Randomization with `{option1|option2}` syntax
- [ ] Clipboard insertion
- [ ] Variables support
- [ ] Forms
- [ ] Rich text (HTML)
- [ ] Fix animation
